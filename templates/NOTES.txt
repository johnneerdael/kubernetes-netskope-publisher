DEPLOYMENT STATUS:
To check the status of your publisher pods:

{{- if include "kubernetes-netskope-publisher.isStatefulSet" . }}
  kubectl get statefulset {{ include "kubernetes-netskope-publisher.fullname" . }} -n {{ .Release.Namespace }}
{{- else }}
  kubectl get daemonset {{ include "kubernetes-netskope-publisher.fullname" . }} -n {{ .Release.Namespace }}
{{- end }}
  kubectl get pods -l app.kubernetes.io/name={{ include "kubernetes-netskope-publisher.name" . }} -n {{ .Release.Namespace }}

LOGS:
To view publisher logs:

  kubectl logs -l app.kubernetes.io/name={{ include "kubernetes-netskope-publisher.name" . }} -n {{ .Release.Namespace }} -c publisher
{{ if include "kubernetes-netskope-publisher.isApiEnrollment" . }}
API enrollment also logs through the publisher container during startup.
{{ else }}
To view enrollment logs:

  kubectl logs -l app.kubernetes.io/name={{ include "kubernetes-netskope-publisher.name" . }} -n {{ .Release.Namespace }} -c enroll
{{- end }}

RESOURCES:
Publisher data is stored in:
{{- if .Values.persistence.enabled }}
  PersistentVolumeClaim: {{ .Values.persistence.existingClaim | default (printf "%s-data" (include "kubernetes-netskope-publisher.fullname" .)) }}
{{- else }}
  EmptyDir (data will be lost on pod restart)
{{- end }}

TROUBLESHOOTING:
If pods are not starting, check:
{{ if include "kubernetes-netskope-publisher.isApiEnrollment" . }}
1. API token Secret exists and contains the expected key:
   kubectl get secret {{ .Values.enrollment.api.existingSecret }} -n {{ .Release.Namespace }}

2. Publisher startup and API enrollment logs:
   kubectl logs <pod-name> -n {{ .Release.Namespace }} -c publisher
{{ else }}
1. Registration token is valid:
   kubectl get secret {{ .Values.registrationToken.existingSecret | default (printf "%s-token" (include "kubernetes-netskope-publisher.fullname" .)) }} -n {{ .Release.Namespace }}

2. Init container enrollment logs:
   kubectl logs <pod-name> -n {{ .Release.Namespace }} -c enroll
{{- end }}

3. Pod events:
   kubectl describe pod <pod-name> -n {{ .Release.Namespace }}

4. Check if nodes meet requirements:
{{ if include "kubernetes-netskope-publisher.isPodNetworking" . }}
   - NET_ADMIN/NET_RAW capabilities
   - /dev/net/tun hostPath character device access
   - Namespace/policy exception for capability and hostPath device use
{{ else }}
   - Host network access
   - Privileged container support
   - NET_ADMIN/NET_RAW capabilities
{{- end }}
